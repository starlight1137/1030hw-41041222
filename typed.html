Typed.js

讓文字如同使用打字機一般呈現
Typed.js官方Github

在<head>中 或<body>前引入script：

 
            <script src="https://unpkg.com/typed.js@2.1.0/dist/typed.umd.js"></script>

範例程式碼:

                
                    //id輸入"typed-output"
                    <h1><span id="typed-output"></span></h1>
                    //於script中設定各項文字特效
                    var typed = new Typed("#typed-output", {
                        strings: ["在此輸入文字"],
                        typeSpeed: 50,     // 打字速度（毫秒）
                        backSpeed: 25,     // 刪除文字速度（毫秒）
                        loop: false        // 是否重複播放
                    });
